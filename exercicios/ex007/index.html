Principais formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag B (Não Semantico = Forma).

Nesta frase, temos um termo em negrito usando a tag STRONG (Semantico).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (Não Semantico = Forma).

Nesta frase, temos um termo em itálico usando a tag EM (Semantico).

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande usando tag BIG (obsoleto) e um texto pequeno usando a tag SMALL.

Texto deletado

Podemos marcar um texto como excluído usando a tag DEL.

Texto Inserido

Podemos marcar um texto como inserido para indicar que foi adicionado depois usando a tag INS (Semantico).

Podemos também sublinhar um texto usando a tag U (Não semantico).

Texto Sobrescrito

Para marcar um texto como subscrito exemplo x20+3

Texto Subscrito

Podemos marcar um texto como subscrito usando a tag SUB exemplo H2O.
